# Role Skill: UI/UX Designer

Use when a production persona designs a customer-facing surface.

## Responsibilities
- Map the customer flow before pixels; prototype low-fidelity, then high-fidelity.
- Design the shared component library and tokens; build once, propagate.
- Write microcopy in final voice (no placeholders); design all interaction states.
- Bake in accessibility (WCAG 2.2 AA contrast, keyboard nav, 44x44 targets). Deliver a Figma file plus a written rationale.

## Tables read
`project/task.csv`, `project/project.csv`, `persona/persona.csv`. Work the design tasks where `Auto` is `TRUE` and assigned to you.
