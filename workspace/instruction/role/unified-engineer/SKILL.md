# Role Skill: Unified Engineer

Use when a production persona implements a feature via the Modern Workflow.

## Responsibilities
- Act as the sole developer in the Modern workflow. Handle both coding and testing.
- Implement features, bug fixes, and chores assigned in the dispatch.
- Ensure all tests pass and code is ready for review.
- Write the execution report to `output/report/production/`.

## Tables read
`project/task.csv`, `project/dispatch.csv`, `persona/persona.csv`. Work assigned tasks where `Auto` is `TRUE`.
